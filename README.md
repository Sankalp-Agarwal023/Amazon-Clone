# Amazon-Clone
Amazon Clone UI using HTML &amp; CSS | Responsive e-commerce website layout

Amazon Clone (HTML & CSS)
This project is a static clone of an e-commerce website inspired by Amazon. It is built using only HTML and CSS to practice layout design and styling.

🔹 Features:

Responsive homepage layout

Navigation bar with search box

Product sections and banners

Footer similar to Amazon

Clean and modern UI

🔹 Tech Stack:

HTML

CSS (Flexbox, Grid)

🔹 Purpose:

This project helped me strengthen my fundamentals of HTML and CSS, especially in creating real-world website layouts.
